# Apple Landing Page

A 3D interactive landing page for MacBook, built with React, Three.js, and GSAP.

## Tech Stack

- React
- Three.js (@react-three/fiber, @react-three/drei)
- GSAP (Animations)
- Tailwind CSS
- Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
